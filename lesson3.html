အပိုင်း ၃: သင့် Python ပတ်ဝန်းကျင်ကို စတင်တပ်ဆင်ခြင်း

ရေးသားသူ Zhorik

Python ပတ်ဝန်းကျင်ကို စတင်တပ်ဆင်ခြင်း

Python ကို သင်၏ ကွန်ပျူတာပေါ်တွင် တပ်ဆင်ပြီး စတင်အသုံးပြုရင်အဆင်ပြေပါလိမ့်မယ်။ အောက်ပါအဆင့်များကိုလိုက်နာပါ:

Python တပ်ဆင်ခြင်း

  1. Python တရားဝင် website ဖြစ်သော python.org မှ Python ကို download ဆွဲပါ။
  2. သင့် operating system အတွက် ကိုက်ညီသော version ကို ရွေးချယ်ပါ။
  3. Download လုပ်ပြီးပါက setup file ကို run လိုက်ပါ။
  4. Install လုပ်နေစဉ်အတွင်း "Add Python to PATH" ကို သေချာစွာ စစ်ပါ။
  5. Install လုပ်ခြင်းကို ပြီးဆုံးပါက terminal သို့မဟုတ် command prompt ကို ဖွင့်ပါ။
  6. python --version ကို ရိုက်ထည့်ပါ။ Python version ကို မြင်ရပါမည်။

Text Editor သို့မဟုတ် IDE ရွေးခြယ်ခြင်း

Python ကို ရေးသားဖို့အတွက် text editor သို့မဟုတ် Integrated Development Environment (IDE) ကို အသုံးပြုနိုင်သည်။ အောက်ပါအတိုင်း ရွေးချယ်နိုင်ပါသည်:

  • Visual Studio Code
  • PyCharm
  • Sublime Text
  • Atom

အထက်ပါ tool များကို တပ်ဆင်ပြီးသည့်အခါ Python code ကို ရေးနိုင်ပါပြီ။

သင့် ပထမဆုံး Python Program

သင့် IDE ကို ဖွင့်ပြီး အောက်ပါ code ကို ရေးပါ:

print("မင်္ဂလာပါ၊ Python!")

ဖိုင်ကို hello.py ဟု သိမ်းပြီး terminal သို့မဟုတ် command prompt မှာ အောက်ပါ command ကို ရိုက်ပါ:

python hello.py

သင့် terminal တွင် မင်္ဂလာပါ၊ Python! ဟု ပြပါမည်။

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. သင့်ကွန်ပျူတာပေါ်တွင် Python ကို download လုပ်ပြီး install လုပ်ပါ။
  2. သင့်အတွက် အကြိုက်ဆုံး IDE သို့မဟုတ် text editor ကို တပ်ဆင်ပါ။
  3. သင့် ပထမဆုံး Python program ကို ရေးပြီး run လုပ်ပါ။ print("မင်္ဂလာပါ၊ Python!") ကို run လုပ်ပါ။
