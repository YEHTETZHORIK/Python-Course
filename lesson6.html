အပိုင်း ၆: Control Flow: if, else, and elif

ရေးသားသူ Zhorik

Control Flow Statements (ထိန်းချုပ်မှု စာကြောင်းများ)

Control flow statements များသည် သင့် program ၏ လုပ်ဆောင်မှုကို ထိန်းချုပ်ရာတွင် အသုံးပြုသည်။ Python တွင် if, else, နှင့် elif statements များကို အသုံးပြု၍ ထိန်းချုပ်နိုင်သည်။

if Statement

Syntax:

if condition:
    statement

ဥပမာ:

age = 18
if age >= 18:
    print("သင့်သည် အရွယ်ရောက်သည်။")

else Statement

Syntax:

if condition:
    statement
else:
    statement

ဥပမာ:

age = 16
if age >= 18:
    print("သင့်သည် အရွယ်ရောက်သည်။")
else:
    print("သင့်သည် အရွယ်မရောက်သေးပါ။")

elif Statement

Syntax:

if condition:
    statement
elif condition:
    statement
else:
    statement

ဥပမာ:

age = 20
if age < 13:
    print("သင့်သည် ကလေးတစ်ယောက် ဖြစ်ပါသည်။")
elif age < 18:
    print("သင့်သည် လူငယ်တစ်ယောက် ဖြစ်ပါသည်။")
else:
    print("သင့်သည် အရွယ်ရောက်သည်။")

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. အသက်ကို သတ်မှတ်ပြီး if statement ကို အသုံးပြုပါ။ သင့်အသက် >= 18 ဖြစ်ပါက "သင့်သည် အရွယ်ရောက်သည်။" ဟု print ပြပါ။
  2. အခြားသော else statement ကို အသုံးပြုပြီး "သင့်သည် အရွယ်မရောက်သေးပါ။" ဟု print ပြပါ။
  3. elif statement ကို အသုံးပြုပြီး အသက်ကို မျှဝေပါ။ < 13 ဖြစ်ပါက "သင့်သည် ကလေးတစ်ယောက် ဖြစ်ပါသည်။" ဟု print ပြပါ၊ < 18 ဖြစ်ပါက "သင့်သည် လူငယ်တစ်ယောက် ဖြစ်ပါသည်။" ဟု print ပြပါ၊ အခြားသော < 18 ဖြစ်ပါက "သင့်သည် အရွယ်ရောက်သည်။" ဟု print ပြပါ။
