Data Types (ဒေတာအမျိုးအစားများ)

Python တွင် မတူညီသော data types များ ရှိသည်။ အောက်ပါအတိုင်း များအတွက် သတ်မှတ်နိုင်သည်:

  • String: စာကြောင်းများ (ဥပမာ: "Hello, World!")
  • Integer: ကိန်းဂဏန်းများ (ဥပမာ: 42)
  • Float: ဒဿမကိန်းများ (ဥပမာ: 3.14)
  • Boolean: True သို့မဟုတ် False (ဥပမာ: True)

Data Types ဥပမာ

name = "Zhorik"  # String
age = 30  # Integer
height = 5.9  # Float
is_student = True  # Boolean

Operators (ဆာင့်လုပ်ငန်းများ)

Operators များသည် data များကို တွက်ချက်ရာတွင် အသုံးပြုသည်။ အောက်ပါအတိုင်း များကို သုံးနိုင်သည်:

Arithmetic Operators

  • + (ပေါင်းခြင်း): a + b
  • - (နုတ်ခြင်း): a - b
  • * (မြောက်ခြင်း): a * b
  • / (ခွဲခြင်း): a / b
  • % (Modulus): a % b

Comparison Operators

  • == (တူညီမှု): a == b
  • != (မတူညီမှု): a != b
  • > (ထက်ကြီး): a > b
  • < (ထက်သေး): a < b
  • >= (ထက်ကြီး သို့မဟုတ် တူညီမှု): a >= b
  • <= (ထက်သေး သို့မဟုတ် တူညီမှု): a <= b

Logical Operators

  • and (နှင့်): a and b
  • or (သို့မဟုတ်): a or b
  • not (မဟုတ်): not a

Operators ဥပမာ

a = 10
b = 5

# Arithmetic Operators
print(a + b)  # 15
print(a - b)  # 5
print(a * b)  # 50
print(a / b)  # 2.0
print(a % b)  # 0

# Comparison Operators
print(a == b)  # False
print(a != b)  # True
print(a > b)  # True
print(a < b)  # False
print(a >= b)  # True
print(a <= b)  # False

# Logical Operators
print(a > 5 and b < 10)  # True
print(a > 15 or b < 10)  # True
print(not(a > 15))  # True

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. String, Integer, Float, နှင့် Boolean data types များကို variable သတ်မှတ်ပါ။
  2. အသုံးပြုမှုကို သုံးပြီး arithmetic operators များကိုစမ်းကြည့်ပါ။
  3. comparison operators များကို အသုံးပြုပြီး value များကို နှိုင်းယှဉ်ကြည့်ပါ။
  4. logical operators များကို အသုံးပြုပြီး logical expressions များကို စမ်းကြည့်ပါ။
