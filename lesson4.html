ရေးသားသူ Zhorik

Python ၏ အခြေခံ Syntax

Python သည် စာကြောင်းအလိုက်အုပ်စုဖွဲ့ရေးရခြင်းမရှိသော programming language တစ်ခုဖြစ်သည်။ Syntax သည် အလွယ်ကူဆုံးဖြစ်ပြီး ဖတ်ရလွယ်ကူသည်။

Variables သတ်မှတ်ခြင်း

Python တွင် variables ကို အလွယ်တကူ သတ်မှတ်နိုင်သည်။ Variable တစ်ခု သတ်မှတ်ရန် အောက်ပါအတိုင်း ရေးပါ:

name = "Zhorik"
age = 30
is_student = True

အထက်ပါ code တွင် name သည် string data type ဖြစ်သည်၊ age သည် integer data type ဖြစ်သည်၊ is_student သည် boolean data type ဖြစ်သည်။

Variables များကို Print ပြရန်

Variable တစ်ခု၏ တန်ဖိုးကို print ပြရန် အောက်ပါအတိုင်း ရေးပါ:

print(name)
print(age)
print(is_student)

သင့် terminal တွင် Zhorik၊ 30 နှင့် True ဟု ပြပါမည်။

String Formatting

String တစ်ခုနှင့် variables များကို ပေါင်းစပ်၍ ရေးရန် အောက်ပါအတိုင်း ရေးနိုင်သည်:

print(f"မင်္ဂလာပါ၊ ကျွန်ုပ်သည် {name} ဖြစ်ပါသည်။ ကျွန်ုပ်၏ အသက်မှာ {age} နှစ် ဖြစ်ပါသည်။ ကျွန်ုပ်သည် ကျောင်းသား ဖြစ်ပါသည်: {is_student}")

အထက်ပါ code မှာ f-string ကို အသုံးပြုပြီး variables များကို string ထဲတွင် ပါဝင်စေရန် ရေးသားထားသည်။

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. သင့်အမည်ကို variable တစ်ခုအနေနဲ့ သတ်မှတ်ပါ။ name = "သင့်အမည်"
  2. သင့်အသက်ကို variable တစ်ခုအနေနဲ့ သတ်မှတ်ပါ။ age = သင့်အသက်
  3. သင့်အမည်နှင့် အသက်ကို print ပြပါ။ print(name) နှင့် print(age) ကို အသုံးပြုပါ။
  4. သင့်အမည်နှင့် အသက်ကို string တစ်ခုထဲတွင် f-string ဖြင့် ပေါင်းစပ်ပြီး print ပြပါ။ print(f"မင်္ဂလာပါ၊ ကျွန်ုပ်သည် {name} ဖြစ်ပါသည်။ ကျွန်ုပ်၏ အသက်မှာ {age} နှစ် ဖြစ်ပါသည်။")
