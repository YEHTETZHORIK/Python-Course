Python အစီအစဉ်ရေးခြင်းနှင့် ပြုလုပ်ခြင်း

Python အစီအစဉ်တစ်ခုကို ရေးခြင်းနှင့် ပြုလုပ်ခြင်းကို စတင်ကြမယ်။

Python အစီအစဉ်ရေးခြင်း

အကြိုက်ဆုံး text editor သို့မဟုတ် IDE ကို ဖွင့်ပြီး အောက်ပါ code ကို ရေးပါ:

print("Hello, World!")

Python အစီအစဉ် ပြုလုပ်ခြင်း

ဖိုင်ကို .py extension နဲ့ သိမ်းပါ၊ ဥပမာ hello.py ။ ထို့နောက် terminal သို့မဟုတ် command prompt ကို ဖွင့်ပြီး သင့်ဖိုင်ကို သိမ်းထားသော directory သို့ သွားပါ။ အောက်ပါ command ကို run ပါ:

python hello.py

သင့်အနေဖြင့် အောက်ပါ output ကို မြင်ရမည်:

Hello, World!

အလွယ်ဆုံး အသုံးပြုနိုင်သော ဥပမာ

အခြေခံ example တစ်ခုမှာ:

print("Python programming အားကောင်းသည်!")

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. သင်၏အမည်ကို print လုပ်ပါ။ print("မင်္ဂလာပါ၊ ကျွန်ုပ်သည် [သင့်အမည်] ဖြစ်ပါသည်")
  2. အခြား string တစ်ခု print လုပ်ပါ။ print("Python သည် အားကောင်းသော programming language တစ်ခုဖြစ်သည်။")
  3. နှစ်ခုထက်ပိုသော string များကို print လုပ်ပါ။ print("မင်္ဂလာပါ") နှင့် print("Python Programming!") ကို run လုပ်ပါ။
