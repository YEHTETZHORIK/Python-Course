အပိုင်း ၇: Loops: for and while

ရေးသားသူ Zhorik

Loops (ရွေ့လျားခြင်းများ)

Loops များသည် တူညီသော code ကို ထပ်ခါထပ်ခါ လုပ်ဆောင်ရန် အသုံးပြုသည်။ Python တွင် for နှင့် while loops များကို အသုံးပြု၍ loop လုပ်နိုင်သည်။

for Loop

Syntax:

for item in iterable:
    statement

ဥပမာ:

fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

အထက်ပါ code သည် "apple", "banana", "cherry" ကို တစ်ခုခြင်းစီ print ပြသည်။

while Loop

Syntax:

while condition:
    statement

ဥပမာ:

count = 0
while count < 5:
    print(count)
    count += 1

အထက်ပါ code သည် 0 မှ 4 အထိ ကိန်းဂဏန်းများကို print ပြသည်။

Nested Loops (ထပ်တူ ရွေ့လျားခြင်းများ)

Loop တစ်ခုထဲတွင် Loop တစ်ခု ထည့်သွင်းနိုင်သည်။

for i in range(3):
    for j in range(2):
        print(f"i: {i}, j: {j}")

အထက်ပါ code သည် i နှင့် j တန်ဖိုးများကို ပြသသည်။

Break နှင့် Continue

break နှင့် continue statements များသည် loop ၏ လုပ်ဆောင်မှုကို ထိန်းချုပ်နိုင်သည်။

for i in range(5):
    if i == 3:
        break
    print(i)  # 0, 1, 2
for i in range(5):
    if i == 3:
        continue
    print(i)  # 0, 1, 2, 4

လေ့ကျင့်ခန်း

အောက်ပါ လေ့ကျင့်ခန်းများကို စမ်းကြည့်ပါ:

  1. for loop ကို အသုံးပြုပြီး 1 မှ 10 အထိ ကိန်းဂဏန်းများကို print ပြပါ။
  2. while loop ကို အသုံးပြုပြီး 0 မှ 4 အထိ ကိန်းဂဏန်းများကို print ပြပါ။
  3. break statement ကို အသုံးပြုပြီး loop ကို ပိတ်ပါ။
  4. continue statement ကို အသုံးပြုပြီး loop ၏ တစ်ခုသော iteration ကို ပြစ်ပါ။
